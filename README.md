# GitHelper

One-click Git setup for username and email.

## Download

Download link: <https://github.com/Leetfs/GitHelper/releases>

### windows

- GitHelper-win32-x64.zip
- GitHelper-Setup.exe
- GitHelper-full.nupkg

### linux

- githelper_amd64.deb
- GitHelper-x86_64.rpm

### macos

- GitHelper-arm64.dmg
- GitHelper-darwin-arm64.zip

## make

```
yarn install
yarn make
```

## start

```
yarn install
yarn start
```
